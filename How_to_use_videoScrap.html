영상 스크랩 확장 프로그램 사용 방법

영상 스크랩 확장 프로그램은 영상의 URL 만 등록하면, 사용자는 별도로 무언가 할 필요가 없습니다.
영상은 항상 크롬의 새 탭 전체에서 재생되므로 해상도 4k 이상을 권장합니다. 4k 이상의 영상을 제공하는 사이트는
대표적으로 "픽사베이" 가 있으며, 해당 사이트의 영상은 무료입니다. 지금부터 안내드릴 영상 스크랩 확장 프로그램
사용 방법은 픽사베이 사이트를 이용한 사용 방법입니다.

첫번째 단계 : 픽사베이 사이트의 동영상 카테고리 접속

픽사베이는 구글의 검색창에서 "픽사베이" 로 검색 후 "비디오" 카테고리를 선택하거나 여기로 접속할 수 있습니다.

두번째 단계 : 4k 영상 찾기

픽사 베이의 비디오 카테고리에 접속 후 나타나는 목록들을 자세히 보십시오. 하나의 항목 왼편 하단에 HD 또는 4K
라고 표시되어 있음을 발견할 수 있습니다. 영상 스크랩 확장 프로그램은 4k 를 적극 권장합니다.

4k 영상 목록만 보고자 한다면 여기를 클릭하세요.

세번째 단계 : 동영상 URL 가져오기

픽사베이의 동영상 목록 중 하나에 마우스를 올려 놓으면 잠시 뒤 영상이 재생됩니다. 이 때 마우스의 오른쪽을 클릭
한 뒤 나타나는 "동영상 주소 복사"를 선택합니다. (이미지를 참조하면 보다 확실히 이해가 됩니다.)

네번째 단계 : 영상 스크랩 확장 프로그램에 영상 등록하기

이제 영상 스크랩 확장 프로그램을 활성하여 [추가] 버튼을 선택 후, 입력 박스 안에 복사한 영상 URL 을 붙여넣은 뒤, 체크 박스를 선택합니다.

이 후 페이지를 새로고침 하거나 새로운 페이지를 열면 등록한 영상이 재생됨을 확인할 수 있습니다.

등록한 영상 삭제

등록한 영상을 삭제하고자 한다면, 체크박스를 해제하고 [삭제] 버튼을 누릅니다.

삭제된 영상 목록은 복원이 불가능합니다.
